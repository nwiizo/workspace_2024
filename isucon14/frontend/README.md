# ISURIDE Frontend

## Stack
- Remix
- tailwind

## Routes

### Top
- **/**
  - ランディングページ

### Client Application
- **/client**
  - ISURIDE利用者用 モバイルクラインアント
- **/client/history**
  - 履歴画面
- **/client/user**
  - ユーザー画面
- **/client/register**
  - ユーザー登録画面
- **/client/login**
  - ユーザーログイン画面

### Owner Application
- **/owner**
  - ISURIDEオーナー向け 管理アプリケーション
- **/owner/sales**
  - 売上確認画面
- **/owner/register**
  - オーナー登録画面
- **/owner/login**
  - オーナーログイン画面

### Simulator
- **/simulator**
  - ISUCON競技者用 アプリ動作シミュレーター
